Instructions:

  • View the full repo for this project at: GitHub
  • The grey boxes can be filled in by the user. When printing pages, the grey backgrounds will be hidden by CSS magic.
  • Text fields must be 'blurred' (click or tab outside of them) for changes to flow.
  • Use a modern browser for best results. Internet Explorer fails on some of the CSS techniques used here.
  • I will eventually make a more mobile-friendly layout, but this one is not very friendly on small screens.
  • Zero out any stress's max to remove the track from your character sheet.
  • Click any circle to toggle an x in it.
  • Click here: to reset your character (changes will be lost, no confirmation).
